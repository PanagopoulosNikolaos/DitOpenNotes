  

Το σύστημα πρωτοκόλλων TCP/IP αποτελεί τη βάση της σύγχρονης δικτύωσης και του διαδικτύου. Παρακάτω παρουσιάζονται τα βασικά σημεία με διαδραστικό τρόπο.

## Επίπεδα του Μοντέλου OSI

**1. Φυσικό επίπεδο:**

- Μετατρέπει τα δεδομένα σε ηλεκτρικό σήμα
- Παρέχει διεπαφή (Interface)

**2. Επίπεδο Σύνδεσης Δεδομένων:**

- Παρέχει διασύνδεση με τον Προσαρμογέα
- Συντηρεί τις λογικές συνδέσεις

**3. Επίπεδο Δικτύου:**

- Υποστηρίζει λογική διευθυνσιοδότηση
- Διαχειρίζεται τη δρομολόγηση

**4. Επίπεδο Μεταφοράς:**

- Παρέχει έλεγχο λαθών
- Παρέχει έλεγχο ροής για το δίκτυο

**5. Επίπεδο Συνόδου:**

- Ορίζει συνόδους μεταξύ εφαρμογών που επικοινωνούν

**6. Επίπεδο Παρουσίασης:**

- Μεταφράζει δεδομένα σε τυπική μορφή
- Διαχειρίζεται κρυπτογράφηση και συμπίεση

**7. Επίπεδο Εφαρμογής:**

- Υποστηρίζει εφαρμογές δικτύου

## Βασικά Επίπεδα TCP/IP

**1. Επίπεδο Πρόσβασης Δικτύου:**

- Παρέχει φυσική διασύνδεση στο δίκτυο
- Μορφοποιεί ανάλογα τα δεδομένα

**2. Επίπεδο Internet:**

- Παρέχει λογική διευθυνσιοδότηση για δρομολόγηση δεδομένων

**3. Επίπεδο Μεταφοράς (TCP):**

- Παρέχει έλεγχο ροής και λαθών
- Προσφέρει υπηρεσίες γνωστοποίησης

**4. Επίπεδο Εφαρμογής:**

- Υποστηρίζει δικτυακές εφαρμογές
- Διαθέτει API για πρόσβαση στο δίκτυο

## Βασικές Λειτουργίες του TCP/IP

- Διαίρεση μηνυμάτων σε αυτοτελή τμήματα δεδομένων
- Διασύνδεση με τον προσαρμογέα δικτύου
- Διευθυνσιοδότηση δεδομένων και υπολογιστή
- Αποστολή δεδομένων στο δίκτυο
- Δρομολόγηση δεδομένων στον προορισμό
- Λήψη δεδομένων
- Εκτέλεση ελέγχου λαθών, ελέγχου ροής και γνωστοποίηση

## Πακέτα Δεδομένων στο TCP/IP

|   |   |   |
|---|---|---|
|Τύπος Πακέτου|Επίπεδο|Περιγραφή|
|message|Επίπεδο Εφαρμογής|Πακέτο δεδομένων που δημιουργείται στο επίπεδο εφαρμογής|
|segment|Επίπεδο Μεταφοράς TCP|Πακέτο δεδομένων από τη δράση του TCP|
|datagram|Επίπεδο Μεταφοράς UDP|Πακέτο δεδομένων από τη δράση του UDP|
|datagram|Επίπεδο Internet|Πακέτο δεδομένων από τη δράση του IP|
|frame|Επίπεδο Πρόσβασης|Πλαίσιο που περιέχει datagram πριν μετατραπεί σε bits|

## Ροή Δεδομένων στην Στοίβα TCP/IP

```Plain
Αποστολή:
Επίπεδο Εφαρμογής → H + message
    ↓
Επίπεδο Μεταφοράς → H + segment/datagram
    ↓
Επίπεδο Δικτύου → H + datagram
    ↓
Επίπεδο Πρόσβασης → H + frame → 101011100010...
```

```Plain
Λήψη:
Επίπεδο Πρόσβασης → H + frame → 101011100010...
    ↑
Επίπεδο Δικτύου → H + datagram
    ↑
Επίπεδο Μεταφοράς → H + segment/datagram
    ↑
Επίπεδο Εφαρμογής → H + message
```

## Πλεονεκτήματα της Αρθρωτής Σχεδίασης

- Μπορεί να προσαρμοστεί εύκολα σε συγκεκριμένο hardware
- Μπορεί να προσαρμοστεί εύκολα σε λειτουργικό σύστημα
