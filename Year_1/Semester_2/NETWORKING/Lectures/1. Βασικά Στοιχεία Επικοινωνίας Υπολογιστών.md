## Βασικοί Ορισμοί

**Δίκτυο Υπολογιστών**  
Δίκτυο είναι ένα σύνολο από υπολογιστές ή συσκευές που μοιάζουν με υπολογιστές, και τα στοιχεία αυτά μπορούν να επικοινωνήσουν μεταξύ τους μέσα από ένα ή περισσότερα κοινά μέσα μετάδοσης.  

**Πρωτόκολλο Δικτύου**  
Πρωτόκολλο Δικτύου είναι ένα σύστημα από κοινούς κανόνες που βοηθά να πραγματοποιηθεί η διαδικασία της μετάδοσης των δεδομένων ανάμεσα στους δικτυακούς πόρους.  

**TCP/IP**  
Το TCP/IP και τα σχετικά του πρωτόκολλα αποτελούν ένα πλήρες σύστημα, το οποίο ορίζει τον τρόπο με τον οποίο θα γίνει η επεξεργασία των δεδομένων, η μετάδοσή τους και η λήψη τους σε δίκτυο που εφαρμόζει τα πρωτόκολλα αυτά.

## Βασικές Λειτουργίες του TCP/IP

|   |   |
|---|---|
|Λειτουργία|Περιγραφή|
|Λογική διευθυνσιοδότηση|Κάθε θύρα (Interface) συσκευής δικτύου έχει μοναδική διεύθυνση IP (32 bits στο IPv4)|
|Δρομολόγηση|Δρομολογητές (Routers) κατευθύνουν δεδομένα στον προορισμό τους και τμηματοποιούν μεγάλα δίκτυα|
|Υπηρεσία επίλυσης ονομάτων|Μετατροπή domain names σε IP addresses (DNS) και NetBIOS ονομάτων σε IP addresses (WINS)|
|Έλεγχος λαθών & ροής|Διασφάλιση αξιοπιστίας στη μεταφορά δεδομένων με χρήση λειτουργιών ελέγχου|
|Υποστήριξη Εφαρμογών|Προσδιορισμός εφαρμογής κάθε πακέτου με χρήση θυρών (ports)|

## Οργανισμοί Τυποποίησης

|   |   |
|---|---|
|Οργανισμός|Ρόλος|
|Internet Advisory Board (IAB)|Ορίζει την πολιτική για το Internet και την εξέλιξη του TCP/IP|
|Internet Engineering Task Force (IETF)|Ορίζει επιτροπές για εξέταση ζητημάτων TCP/IP|
|Internet Research Task Force (IRTF)|Χρηματοδοτεί έρευνα σχετική με το Internet|
|Internet Corporation for Assigned Names and Numbers (ICANN)|Εκχωρεί domain names και IP addresses|
|InterNIC|Διατηρεί λίστα οργανισμών που παρέχουν Domain Names|

## RFC (Requests For Comment)

Τα RFC είναι επίσημα τεχνικά έγγραφα που παρέχουν τεχνικές πληροφορίες και τυποποιήσεις που αφορούν θέματα του TCP/IP, του Internet και άλλων πρωτοκόλλων.

```Mermaid

    A --> C[Τυποποιήσεις]
    A --> D[Βιβλιοθήκη τεκμηρίωσης]
    B --> E[TCP/IP]
    B --> F[Internet]
    C --> G[Πρωτόκολλα επικοινωνίας]
```
