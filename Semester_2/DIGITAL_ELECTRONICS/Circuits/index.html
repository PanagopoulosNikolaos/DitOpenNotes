Fundamental Definitions

Flip-Flop

A bistable multivibrator circuit that can store one bit of information and change state on clock edges.

Shift Register

A cascade of flip-flops that can move data left or right, used for serial data transfer and temporary storage.

Counter

A sequential circuit that cycles through a predetermined sequence of states, typically binary counting.

Decoder

A combinational circuit that converts binary coded input to a one-hot output representation.

Interactive Glossary

Setup Time

The minimum time data must be stable before the clock edge.

Critical timing parameter that ensures reliable data capture in sequential circuits.

Hold Time

The minimum time data must remain stable after the clock edge.

Prevents data corruption during the transition period of flip-flops.

Propagation Delay

Time delay between input change and corresponding output change.

Important factor in determining maximum operating frequency of circuits.

Metastability

Unstable state when flip-flop inputs violate timing requirements.

Can cause unpredictable behavior and must be avoided in reliable designs.

Shift Register Applications

  • Serial Communication: Converting parallel data to serial for transmission
  • Delay Lines: Creating time delays in digital signals
  • Sequence Generation: Producing specific bit patterns
  • Temporary Storage: Buffering data during processing

Key Formulas

Serial Transfer Time: T = n × Tclk

Where n is the number of bits and Tclk is the clock period

Maximum Frequency: fmax = 1 / (tpd + tsetup)

Limited by propagation delay and setup time requirements

Decoder Theory

A decoder is a combinational logic circuit that converts n binary inputs into 2^n unique outputs, where only one output is active (high) for each input combination.

Boolean Expressions

  • Y₁ = Ā·B
  • Y₂ = A·B̄
  • Y₃ = A·B
